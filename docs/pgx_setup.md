# PGX setup notes

このアプリはPGX上で動かす前提です。

## 1. Clone

```bash
git clone https://github.com/onotakatoshi/Musubi_LoRA_Factory.git
cd Musubi_LoRA_Factory
```

## 2. Python environment

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 3. Settings

```bash
cp configs/settings.example.toml configs/settings.toml
```

`configs/settings.toml` をPGX環境に合わせて編集します。

特に重要:

```toml
[musubi]
repo_path = "/home/ono/musubi-tuner"
python_path = "/home/ono/musubi-tuner/.venv/bin/python"

[paths]
datasets_dir = "/home/ono/datasets/lora"
outputs_dir = "/home/ono/outputs/lora"
comfyui_loras_dir = "/home/ono/ComfyUI/models/loras"

[model_paths]
wan_vae = "..."
wan_t5 = "..."
wan_dit = "...low_noise..."
wan_dit_high_noise = "...high_noise..."
```

## 4. Start GUI

```bash
python app/main.py
```

Open:

```text
http://PGX_IP_ADDRESS:7865
```

or locally on PGX:

```text
http://localhost:7865
```

## 5. First test dataset

Recommended first dataset:

```text
512x512 image files: 20-100
caption txt: generated by app
LoRA type: eye
Target model: wan2.2
Rank/Alpha: 16/16
Epochs: 10
Learning rate: 0.00005
```

## 6. Development rule

First run should be command preview only. Do not start real training until:

- musubi-tuner path is confirmed
- VAE path is confirmed
- T5 path is confirmed
- Wan2.2 low/high DiT paths are confirmed
- dataset.toml is generated correctly
